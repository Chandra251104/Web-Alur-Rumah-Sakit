# Sistem Informasi Rumah Sakit

Aplikasi manajemen pelayanan pasien berbasis web menggunakan CodeIgniter 3.

## Fitur
- Registrasi pasien
- Pemeriksaan pasien
- Diagnosa dokter
- Pembuatan resep
- Pembayaran
- Pengambilan obat di farmasi

## Teknologi
- PHP
- CodeIgniter 3
- MySQL
- Bootstrap 5

## Alur Sistem
Pasien → Registrasi → Pemeriksaan → Diagnosa → Resep → Pembayaran → Farmasi
